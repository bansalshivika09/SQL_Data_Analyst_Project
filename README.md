#SQL Data Analyst Portfolio Project

## Project Overview

This project demonstrates fundamental and advanced SQL concepts using a bookstore sales database. The objective of the project is to practice data analysis techniques and SQL query writing on real-world style datasets.

The project uses three datasets:

* **Books** – Contains information about books such as titles, authors, categories, and prices.
* **Customers** – Contains customer-related information.
* **Orders** – Contains order and sales transaction details.

The datasets were imported into **MySQL Workbench**, where various SQL queries were written and executed to analyze the data.

## Skills Demonstrated

### Basic SQL Concepts

* SELECT statements
* WHERE clauses
* ORDER BY
* DISTINCT
* Filtering records
* Sorting data

### Intermediate and Advanced SQL Concepts

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* Aggregate Functions (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
* GROUP BY
* HAVING Clause
* Multi-table analysis
* Data summarization and reporting
